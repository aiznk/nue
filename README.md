# Nue

Nue is a lightweight JavaScript framework for frontend development.

```js
import * as nue from "../nue.js"

class Root extends nue.Root {
	constructor () {
		super()
		this.msg = new nue.P()
		this.msg.setText('Hello, World!')
		this.add(this.msg)
	}
}

document.addEventListener('DOMContentLoaded', () => {
	let root = new Root()
	root.mount('#app')
})
```

## Design

Nue is component-oriented. Components are used to build a tree structure, and events generated in leaf components can be propagated to parent components. This makes it possible to centralize all events generated by child components (such as clicks and inputs) within parent components.

## Mounting Components

Components can be mounted onto DOM elements. The component will be rendered inside the mounted element.

```js
class Root extends nue.Root {
	constructor () {
		super()
	}
}

let root = new Root()
root.mount('#my-app')
```

## Root Component

The Root component is not significantly different from other components; the only difference is its name. Like other components, the Root component inherits from the `Component` class.

By default, the Root component uses a `div` element, but this can be changed by passing a tag name and attributes to `super`.

```js
class Root extends nue.Root {
	constructor () {
		super('span', {
			class: 'my-root',
		})
	}
}
```

## Referencing a Component's DOM Element

You can access the DOM element owned by a component through the component's `elem` property.

```js
class MyDiv extends nue.Div {
	constructor () {
		super()

		let div = this.elem
		div.textContent = 'text text'
	}
}
```

## Changing Component Attributes

To change the attributes of components such as `Div` or `Span`, modify the constructor arguments.

```js
class MyDiv extends nue.Div {
	constructor () {
		super({
			class: 'my-class',
			id: 'my-id',
		})
	}
}
```

Alternatively, you can modify attributes using the `setAttr` method.

```js
class MyDiv extends nue.Div {
	constructor () {
		super()

		this.setAttr('class', 'my-class')
		this.setAttr('id', 'my-id')
	}
}
```

## Getting and Setting Text or Values

Use the `getText` and `setText` methods to get or set text content. These are equivalent to accessing `this.elem.textContent`.

Use the `getValue` and `setValue` methods to get or set values. These are equivalent to accessing `this.elem.value`.

```js
class MyP extends nue.P {
}

class MyInput extends nue.Input {
}

let p = new MyP()
p.setText('Hello, World!')
alert(p.getText())

alert(p.elem.textContent)

let input = new MyInput()
input.setValue('default value')
alert(input.getValue())

alert(input.elem.value)
```

## Adding Child Components

Child components are added using the parent component's `add` method.

```js
class Root extends nue.Root {
	constructor () {
		super()

		this.child = new nue.P()
		this.add(this.child) // child is child component of root
	}
}
```

## Event Propagation and Handling

To capture events, configure the component, handle the events, and `emit` them to the parent component.

```js
class MyInput extends nue.Input {
	constructor () {
		super({}, {
			events: ['click', 'keydown'],
		})
	}

	onClick (ev) {
		this.emit('clicked!', ev) // emit to parent component
	}

	onKeydown (ev) {
		this.emit('keydown!', ev) // emit to parent component
	}
}
```

Events emitted with `emit` are recursively propagated through connected parent nodes. In other words, all events are automatically centralized in parent components without requiring additional code.

To capture propagated events, override the `receive` method.

```js
class Parent extends nue.Div {
	constructor () {
		super()

		this.input = new MyInput()
		this.add(this.input)
	}

	receive (key, val) {
		switch (key) {
		case 'clicked!': console.log(val); break
		case 'keydown!': console.log(val); break
		}
	}
}
```

## Building Trees with Templates

Normally, component trees are constructed by adding child components using the `add` method.

However, there are situations where it is more convenient to build tree structures in an HTML-like manner. In such cases, use the `template` method.

```js
class Root extends nue.Root {
	constructor () {
		super()

		this.myInput = new nue.Input()

		this.template(`
			<div class="my-class">
				<MyInput />
			</div>
		`, {
			components: {
				MyInput: this.myInput,
			}
		})
	}
}
```

The `template` method allows you to describe components and HTML elements similarly to HTML. By specifying components in the `components` argument, component names inside the template are replaced with the corresponding component instances.
